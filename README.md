#### 附
brew 上面我没有找到好的wol工具,于是就自己写了一个.
- 大家可以通过修改一下代码进行绑定自定义的地址和端口
```
#define URL "localhost"
#define PORT 9
```
- 大家也可以进行参数传递 
```
wol [url] [port]
```

### 知识点汇总
```
    /**
     * UDP数据包，端口不限，数据内容是一个有着特定格式的数据包：
     * Magic Packet，其格式为：6个0xFF加16个目标网卡MAC地址，
     * 因此该Magic Packet总长度为。6+16*6+6=108字节
     */
```
